# 🎙️ AI vs Human Voice Detector

An AI-powered system that detects whether a given voice sample is **human** or **AI-generated** using machine learning and audio signal processing.

---

## 🚀 Features
- 🎧 Upload audio files (WAV/MP3)
- 🤖 Classifies voice as **Human** or **AI-generated**
- 📊 Uses audio features like MFCC & spectral analysis
- ⚡ Fast and efficient predictions
- 🌐 Simple web interface (Flask)

---

## 🧠 Tech Stack
- Python
- Machine Learning
- Librosa (Audio processing)
- Scikit-learn / TensorFlow (Model)
- Flask (Web app)
